

## Dependencies

### Libxsmm::

* Required:: GCC must be `8.3.0` or above

```bash
	git clone https://github.com/hfp/libxsmm.git
	make realclean
	make -j
```

* Set LIBXSMM Path::

```bash
	export LIBXSMM_ROOT=/path/to/libxsmm
```

## mlpcell Installation::

```python
	python setup.py install
```
